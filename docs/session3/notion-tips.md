## 노션 첫 화면, 여기가 어디지?

처음 로그인하면 화면이 낯설어요. 구조부터 파악하면 다 보입니다.

```
┌─────────────────────────────────────────────┐
│  사이드바 (왼쪽)     │  페이지 영역 (오른쪽) │
│                      │                       │
│  🔍 검색             │                       │
│  🏠 홈               │   여기에 내용 작성    │
│  📥 받은 편지함      │                       │
│  ─────────────       │                       │
│  📁 팀 페이지들      │                       │
│  📄 내 페이지들      │                       │
│  ─────────────       │                       │
│  + 새 페이지         │                       │
└─────────────────────────────────────────────┘
```

| 영역 | 설명 |
|------|------|
| **검색 (🔍)** | `Ctrl+P` / `Cmd+P` — 모든 페이지 빠른 검색 |
| **홈 (🏠)** | 최근 열어본 페이지, 즐겨찾기 모음 |
| **받은 편지함** | 팀원이 나를 @멘션하거나 코멘트 달면 알림 |
| **사이드바 페이지 목록** | 왼쪽에 있는 페이지 이름들 = 폴더 구조 |
| **+ 새 페이지** | 사이드바 하단 — 새 페이지 만들기 |

!!! tip "사이드바가 안 보일 때"
    화면 왼쪽 가장자리에 마우스를 갖다 대면 사이드바가 나타납니다.
    고정하려면 사이드바 상단 `«` 아이콘 클릭.

## 블록 완전 정복

### 블록이란?

노션의 모든 내용은 **블록 단위**로 이루어져 있어요.
텍스트 한 줄, 이미지 하나, 표 하나 — 각각이 블록입니다.

### 블록 만들기

```
방법 1: 빈 줄에서 / 입력 → 메뉴에서 선택
방법 2: Enter 눌러 새 줄 → 바로 타이핑 (일반 텍스트)
방법 3: 기존 내용 아래 빈 곳 클릭 → / 입력
```

### 블록 선택·이동·삭제

블록 왼쪽에 마우스를 올리면 **`⠿` 핸들**이 나타납니다.

| 동작 | 방법 |
|------|------|
| **이동** | `⠿` 핸들을 드래그해서 원하는 위치로 |
| **선택** | `⠿` 클릭 → 블록 전체 선택 |
| **삭제** | 블록 선택 후 `Delete` 또는 `Backspace` |
| **복제** | `⠿` 클릭 → **복제** 또는 `Ctrl+D` |
| **블록 메뉴** | `⠿` 클릭 → 색상·전환·삭제 등 옵션 |
| **여러 블록 선택** | `Shift` 누른 채 여러 블록 클릭 |

!!! tip "블록 타입 바꾸기"
    이미 작성한 내용의 타입을 바꾸고 싶을 때:
    `⠿` 클릭 → **"전환"** → 원하는 타입 선택
    (예: 일반 텍스트 → 체크리스트로 한 번에 변환)

### 블록 종류 전체 목록

=== "텍스트·제목"

    | 명령어 | 결과 |
    |--------|------|
    | `/text` | 일반 텍스트 |
    | `/h1` | 큰 제목 |
    | `/h2` | 중간 제목 |
    | `/h3` | 작은 제목 |
    | `/quote` | 인용구 (왼쪽 선 강조) |
    | `/callout` | 콜아웃 박스 (강조 메모) |

=== "목록"

    | 명령어 | 결과 |
    |--------|------|
    | `/bullet` | 글머리 목록 (•) |
    | `/number` | 번호 목록 (1. 2. 3.) |
    | `/todo` | 체크박스 목록 |
    | `/toggle` | 토글 (접었다 펼치는 목록) |

=== "레이아웃"

    | 명령어 | 결과 |
    |--------|------|
    | `/divider` | 구분선 |
    | `/columns` | 다단 레이아웃 (2컬럼, 3컬럼) |
    | `/page` | 하위 페이지 삽입 |

=== "미디어"

    | 명령어 | 결과 |
    |--------|------|
    | `/image` | 이미지 업로드 또는 URL |
    | `/video` | 동영상 삽입 |
    | `/file` | 파일 첨부 |
    | `/embed` | 외부 콘텐츠 삽입 |

=== "데이터"

    | 명령어 | 결과 |
    |--------|------|
    | `/table` | 단순 표 |
    | `/database` | 데이터베이스 (스프레드시트형) |
    | `/board` | 칸반 보드 |
    | `/calendar` | 캘린더 뷰 |

## 텍스트 꾸미기

### 마크다운 단축키

타이핑하면서 바로 적용돼요.

| 입력 | 결과 |
|------|------|
| `**텍스트**` | **굵게** |
| `*텍스트*` | *기울임* |
| `` `코드` `` | `인라인 코드` |
| `~~텍스트~~` | ~~취소선~~ |
| `[제목](url)` | 하이퍼링크 |

### 선택 후 서식 적용

텍스트를 드래그해서 선택하면 위에 **서식 도구막대**가 뜹니다.

```
[ B ] [ I ] [ U ] [ S ] [ 코드 ] [ 링크 ] [ 색상 ] ...
  굵게  기울  밑줄  취소선  코드    링크    색상
```

| 단축키 | 효과 |
|--------|------|
| `Ctrl+B` / `Cmd+B` | 굵게 |
| `Ctrl+I` / `Cmd+I` | 기울임 |
| `Ctrl+U` / `Cmd+U` | 밑줄 |
| `Ctrl+Shift+S` | 취소선 |
| `Ctrl+E` / `Cmd+E` | 인라인 코드 |
| `Ctrl+K` / `Cmd+K` | 링크 삽입 |

### 텍스트 색상·배경색

```
텍스트 선택 → 서식 막대 → A (색상) 클릭
→ 텍스트 색상 또는 배경 색상 선택
```

!!! tip "강조 효과 조합 추천"
    - 중요 수치: **굵게** + 노란 배경
    - 주의사항: 빨간 텍스트
    - 완료 항목: ~~취소선~~ + 회색 텍스트

## 토글 블록 — 접었다 펼치는 목록

FAQ나 긴 내용을 깔끔하게 정리할 때 최고예요.

### 만드는 법

```
/toggle 입력 → 제목 작성 → Enter
→ 아래 들여쓰기 된 곳에 내용 작성
→ 클릭하면 접히고 펼쳐짐
```

### 활용 예시

```
▶ 금식이 꼭 필요한가요?
  └ 혈액검사와 위내시경이 포함된 경우 전날 자정부터 금식이 필요합니다.
    단, 물과 필수 약물은 복용 가능합니다.

▶ 검진 결과는 언제 나오나요?
  └ 기본 혈액·소변 검사는 당일, 조직 검사가 포함된 경우 5~7일 소요됩니다.

▶ 혈압약을 먹어도 되나요?
  └ 혈압약은 소량의 물과 함께 복용하셔도 됩니다. 다른 약물은 담당자에게 문의해 주세요.
```

→ FAQ 페이지를 이렇게 만들면 깔끔하게 정리됩니다.

!!! tip "토글 안에 토글"
    토글 블록 안에 또 토글을 넣을 수 있어요.
    카테고리 → 세부 항목으로 2단계 구조 만들기 가능.

## 2컬럼 레이아웃 — 나란히 배치

### 만드는 법

**방법 1: 드래그**
```
블록의 ⠿ 핸들을 잡고
→ 옆에 있는 다른 블록 오른쪽 끝으로 드래그
→ 파란 세로선이 나타나면 놓기
→ 2컬럼 완성
```

**방법 2: 명령어**
```
/columns 입력 → 2 columns 또는 3 columns 선택
```

### 활용 예시

```
┌──────────────────┬──────────────────┐
│  ✅ 검진 전 챙길 것  │  ❌ 하지 말 것      │
│                  │                  │
│  • 신분증 지참     │  • 아침 식사 금지  │
│  • 편한 복장      │  • 음주 금지 (2일)  │
│  • 검진 예약 확인  │  • 격렬한 운동     │
└──────────────────┴──────────────────┘
```

## 검색 기능 — 어디 있지? 바로 찾기

노션에 내용이 쌓일수록 검색이 생명이에요.

### 빠른 검색

```
Ctrl+P (Windows) / Cmd+P (Mac)
→ 검색창 바로 열림
→ 페이지 이름 또는 내용 입력
→ Enter로 바로 이동
```

### 검색 팁

| 상황 | 검색 방법 |
|------|----------|
| 페이지 이름으로 | 제목 일부만 입력해도 됨 |
| 내용으로 | 안에 있는 텍스트 그대로 검색 |
| 최근 페이지 | 검색창 열면 최근 방문 목록 자동 표시 |
| 팀원 멘션 | `@이름` 으로 특정 멘션 검색 |

!!! tip "즐겨찾기 활용"
    자주 쓰는 페이지는 즐겨찾기 해두세요.
    페이지 우상단 `⋯` → **"즐겨찾기에 추가"**
    → 사이드바 상단 ⭐ 목록에 고정됩니다.

## @멘션 — 연결의 핵심

`@` 를 입력하면 팀원·페이지·날짜를 연결할 수 있어요.

| 입력 | 결과 |
|------|------|
| `@이름` | 팀원 멘션 (알림 발송) |
| `@페이지이름` | 다른 노션 페이지 링크 |
| `@오늘` | 오늘 날짜 삽입 |
| `@내일` | 내일 날짜 삽입 |
| `@2026-07-01` | 특정 날짜 삽입 |

!!! tip "@멘션 실전 활용"
    회의록에서 담당자 지정:
    ```
    - 재검 안내 문자 발송: @홍길동 (마감: @2026-06-20)
    ```
    → 홍길동에게 알림이 가고, 날짜가 클릭 가능한 링크로 표시됩니다.

## 노션 AI 상세 활용

### 노션 AI 여는 방법

```
방법 1: 빈 줄에서 스페이스바 → AI 메뉴 열림
방법 2: 텍스트 선택 후 → AI 버튼 클릭
방법 3: / 입력 → "AI로 작성" 선택
```

### 상황별 AI 활용법

=== "새 내용 생성"

    빈 줄에서 스페이스바 누르면:

    | AI 명령 | 결과 |
    |---------|------|
    | `브레인스토밍해줘` | 아이디어 목록 |
    | `개요 만들어줘` | 구조화된 목차 |
    | `계속 써줘` | 기존 내용 이어서 작성 |
    | `블로그 포스트 써줘` | 긴 형식 글 |

=== "기존 내용 개선"

    텍스트를 선택하고 AI 클릭:

    | AI 명령 | 결과 |
    |---------|------|
    | `요약해줘` | 핵심 내용 압축 |
    | `더 길게` | 내용 확장 |
    | `더 짧게` | 내용 축약 |
    | `더 쉽게 설명해줘` | 쉬운 말로 변환 |
    | `전문적으로 바꿔줘` | 공식 문서 스타일로 |
    | `맞춤법 교정` | 오탈자·문법 수정 |
    | `번역: 영어로` | 다른 언어로 번역 |

=== "정리·분석"

    | AI 명령 | 결과 |
    |---------|------|
    | `표로 정리해줘` | 산문 → 표 변환 |
    | `액션 아이템 뽑아줘` | 할 일 목록 추출 |
    | `주요 포인트 뽑아줘` | 핵심 3~5가지 |
    | `FAQ 만들어줘` | Q&A 형식으로 변환 |

### 노션 AI 실전 예시

**예시 1 · 회의 메모 → 회의록**
```
회의 때 대충 메모한 내용을 붙여넣고
→ 텍스트 전체 선택
→ AI → "정리해줘"
→ 깔끔한 회의록으로 자동 변환
```

**예시 2 · ChatGPT 안내문 → 노션 최적화**
```
ChatGPT가 만든 안내문을 노션에 붙여넣고
→ 전체 선택 → AI → "표로 정리해줘"
→ 항목별 표 형태로 자동 변환
```

**예시 3 · 긴 안내문 → 핵심 요약**
```
긴 검진 안내문 선택
→ AI → "수검자가 꼭 알아야 할 핵심 3가지만 뽑아줘"
→ 요약본 자동 생성
```

!!! warning "노션 AI 무료 한도"
    노션 AI는 **월 20회까지 무료** (이후 유료 전환 필요)
    중요한 작업에 집중해서 사용하고, 기본 편집은 직접 하는 게 유리해요.

## 데이터베이스(DB) 기초 실습

### DB란 무엇인가

일반 표보다 훨씬 강력한 "스마트 표"예요.
같은 데이터를 **여러 모양**으로 볼 수 있습니다.

```
같은 데이터  →  표로 보기 (스프레드시트)
            →  칸반으로 보기 (상태별 카드)
            →  캘린더로 보기 (날짜별)
            →  갤러리로 보기 (카드형)
```

### 검진 안내문 DB 직접 만들기

**Step 1.** 새 페이지에서 DB 생성
```
/table 입력 → "전체 페이지 테이블" 선택
```

**Step 2.** 속성(컬럼) 추가
```
열 제목 클릭 → 이름 변경
+ 버튼 클릭 → 새 속성 추가
```

**검진 안내문 DB 추천 속성:**

| 속성명 | 타입 | 용도 |
|--------|------|------|
| 문서명 | 제목 | 안내문 이름 |
| 검진 항목 | 선택 | 혈압, 혈당, 중성지방 등 |
| 대상 | 텍스트 | 수치 범위 등 |
| 담당 부서 | 선택 | 간호팀, 원무팀 등 |
| 작성일 | 날짜 | 작성한 날짜 |
| 상태 | 상태 | 초안, 검토중, 완료 |
| 담당자 | 사람 | 팀원 지정 |

**Step 3.** 뷰 추가하기
```
왼쪽 상단 "+ 뷰 추가" 클릭
→ 보드 / 캘린더 / 갤러리 선택
```

!!! success "DB 활용의 진짜 힘"
    - **상태 필터:** "완료" 문서만 보기
    - **담당자 필터:** 내가 담당한 문서만 보기
    - **날짜 정렬:** 최신 순으로 정렬
    - **검진 항목별 그룹:** 항목별로 묶어서 보기

## 공유 완전 가이드

### 공유 방법 3가지

=== "팀원 초대 (내부 공유)"

    ```
    페이지 우상단 "공유" 클릭
    → "사람 또는 그룹 추가" 입력창
    → 이메일 입력
    → 권한 선택 후 초대
    ```

    **권한 종류:**

    | 권한 | 할 수 있는 것 |
    |------|-------------|
    | **전체 접근** | 편집·공유·삭제 모두 가능 |
    | **편집** | 내용 수정 가능, 공유 불가 |
    | **댓글** | 댓글만 달 수 있음 |
    | **보기** | 읽기만 가능 |

=== "링크 공유 (외부 공개)"

    ```
    공유 → "링크 게시" 토글 ON
    → 링크 복사 → 카톡·메일로 전송
    ```

    !!! warning "외부 공개 주의"
        "링크 게시"를 ON하면 **로그인 없이도 누구나 볼 수 있어요.**
        내부 업무 문서는 OFF 상태를 유지하세요.

=== "워크스페이스 공유"

    ```
    공유 → "워크스페이스의 모든 사람"
    → 같은 워크스페이스 팀원 전체에게 공개
    ```

### 페이지 잠금 (실수 방지)

```
페이지 우상단 ⋯ → "잠금"
→ 수정 불가 상태로 고정
→ 다시 ⋯ → "잠금 해제"로 편집 재개
```

## 노션 실전 꿀팁 모음

### 꿀팁 1 · 페이지 아이콘과 커버로 한눈에 구분

```
페이지 제목 위에 마우스 올리기
→ "아이콘 추가" 클릭 → 이모지 선택
→ "커버 추가" 클릭 → 이미지 선택
```

부서별로 아이콘을 다르게 하면 한눈에 구분됩니다:
- 🏥 우리 팀 전체
- 💉 간호팀 자료
- 📋 원무팀 자료
- 📢 공지·마케팅

---

### 꿀팁 2 · 사이드바에서 페이지 순서 바꾸기

```
사이드바에서 페이지 이름 드래그
→ 원하는 위치로 이동
```

중요한 페이지를 위로 올려두면 매번 찾을 필요 없어요.

---

### 꿀팁 3 · 링크드 뷰 — 같은 DB를 여러 곳에

같은 데이터베이스를 다른 페이지에서도 볼 수 있어요.

```
/linked view 입력
→ 기존 DB 선택
→ 이 페이지에서 원하는 뷰로 보기
```

예: 간호팀 페이지에서는 간호팀 문서만, 원무팀 페이지에서는 원무팀 문서만 필터해서 보기.

---

### 꿀팁 4 · 템플릿 버튼

반복적인 형식을 버튼 하나로 찍어내기.

```
/template 입력
→ 버튼 이름 입력 (예: "새 안내문 추가")
→ 템플릿 내용 작성
→ 버튼 클릭할 때마다 해당 형식이 자동 생성
```

---

### 꿀팁 5 · 노션 웹 클리퍼

웹에서 찾은 정보를 노션에 바로 저장.

```
크롬 확장 프로그램 "Notion Web Clipper" 설치
→ 저장하고 싶은 웹페이지에서 아이콘 클릭
→ 저장할 노션 페이지 선택
→ 바로 저장
```

## 자주 하는 실수 & 해결법

!!! danger "이런 실수, 저도 처음엔 했어요"

=== "실수 1 · 내용이 사라졌어요"

    **원인:** 실수로 삭제하거나 다른 페이지로 이동됨

    **해결:**
    ```
    Ctrl+Z (실행 취소) — 방금 삭제한 경우
    페이지 우상단 ⋯ → "히스토리" — 이전 버전 복원
    사이드바 하단 "휴지통" — 삭제된 페이지 복구
    ```

=== "실수 2 · 페이지가 너무 복잡해졌어요"

    **원인:** 한 페이지에 모든 걸 넣어서 길어짐

    **해결:**
    내용을 선택 → `⠿` → **"페이지로 전환"**
    → 하위 페이지로 분리됩니다.

=== "실수 3 · 팀원이 편집할 수 없다고 해요"

    **원인:** 공유 시 권한을 "보기"로 설정함

    **해결:**
    ```
    공유 → 해당 팀원 이름 옆 권한 드롭다운
    → "편집"으로 변경
    ```

=== "실수 4 · 외부인이 내부 문서를 봤어요"

    **원인:** 공유 시 "링크 게시"를 실수로 ON

    **해결:**
    ```
    공유 → "링크 게시" 토글 OFF
    → 이후 해당 링크는 접근 불가
    ```

=== "실수 5 · 블록이 엉뚱한 곳에 들어갔어요"

    **원인:** 드래그 중 실수로 다른 블록 안으로 이동

    **해결:**
    ```
    Ctrl+Z 로 바로 되돌리기
    또는 ⠿ 핸들로 다시 드래그해서 원래 위치로
    ```
