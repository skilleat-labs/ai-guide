# 이미지 잘 뽑는 법 총정리

> 수업에서 나온 이미지 프롬프트 팁을 한 페이지에 모았습니다.
> 나중에 혼자 이미지 만들 때 꺼내보세요.

---

## 핵심 공식 — 이미지 4요소

| 요소 | 내용 | 예시 |
|------|------|------|
| **무엇을** | 그릴 대상 | "금식 안내, 물컵 든 사람" |
| **스타일** | 그림체·분위기 | "따뜻한 플랫 일러스트" |
| **색감** | 색·무드 | "파스텔 파란 톤" |
| **용도** | 크기·형태 | "가로 안내문용 16:9" |

```
주제 + 스타일 + 색감 + 비율 순서로 한 문장에 묶기

예) "건강검진 금식 안내, 플랫 일러스트, 파스텔 파란 톤, 가로 16:9"
```

## 수정할 때 — 이것만 지키면 안 흔들립니다

!!! warning "drift 주의"
    한 번에 여러 곳을 고치면 이미지 전체가 바뀝니다.
    **한 번에 하나씩, 유지 조건은 매번 다시.**

=== "나쁜 예 ❌"

    ```
    "옷을 후드티로 바꿔줘"
    ```
    → 얼굴·배경·조명까지 같이 바뀜

=== "좋은 예 ✅"

    ```
    "옷만 회색 후드티로 바꿔줘.
    얼굴·표정·머리·배경·포즈는 그대로 유지."
    ```

### 수정 요청 표현 모음

| 상황 | 입력 |
|------|------|
| 너무 어두움 | "더 밝게. 구성·인물은 유지해줘" |
| 글자 넣고 싶음 | "'금식 8시간' 상단에 굵게. 오타·추가 글자 금지" |
| 색이 마음에 안 듦 | "파란 톤만 초록으로 바꿔줘. 나머지는 그대로" |
| 스타일이 다름 | "플랫 일러스트로 다시 그려줘" |
| 처음부터 다시 | "이 느낌 말고, 더 미니멀하게 다시" |

## 장르별 체크리스트

=== "텍스트 넣을 때"

    - [ ] 문구를 **따옴표**로 감싸서 입력
    - [ ] 글꼴·크기·위치 명시
    - [ ] `"오타·번역·추가 글자 금지"` 반드시 포함

    ```
    상단에 '검사 전 8시간 금식' 텍스트, 굵게, 한국어 그대로,
    오타·추가 글자 절대 금지.
    ```

=== "인물이 있을 때"

    - [ ] 얼굴형·눈매·표정 유지 명시
    - [ ] 헤어·안경·피부톤도 고정
    - [ ] 바꿀 것만 콕 집어서

    ```
    옷만 흰 가운으로 바꿔줘.
    얼굴형·눈매·표정·머리·피부톤은 그대로 유지.
    ```

=== "실사 느낌"

    - [ ] `"카메라로 찍은 듯"` 명시
    - [ ] 렌즈·조명 조건 추가
    - [ ] 피부 질감·옷감 디테일 명시

    ```
    사실적인 사진 스타일, 35mm 렌즈, 자연광, 배경 살짝 흐리게.
    ```

=== "인포그래픽"

    - [ ] 단계 수와 각 단계 이름 명시
    - [ ] 들어갈 텍스트 따옴표로
    - [ ] 화살표·아이콘 포함 여부
    - [ ] `"10초 안에 구조 이해"` 기준으로

    ```
    흰 배경, 4단계 화살표 인포그래픽.
    1단계 '접수', 2단계 '문진', 3단계 '검사', 4단계 '결과'.
    각 단계에 아이콘, 파란 계열, 가로형.
    작은 글자·복잡한 배경 금지.
    ```

## 완성 프롬프트 예시 모음

=== "금식 안내 이미지"

    ```
    건강검진 금식 안내 가로 이미지,
    물컵을 든 사람이 있는 심플한 장면,
    따뜻한 플랫 일러스트, 파스텔 파란 톤,
    흰 배경, 16:9 비율.
    ```

=== "검진 절차 인포그래픽"

    ```
    건강검진 절차 4단계 인포그래픽.
    1단계 '접수', 2단계 '문진', 3단계 '검사', 4단계 '결과'.
    각 단계에 심플한 아이콘, 화살표로 흐름 연결,
    차분한 파란 계열, 가로형 흰 배경.
    10초 안에 구조 이해되는 깔끔한 플랫 디자인.
    ```

=== "글자 들어간 안내 카드"

    ```
    건강검진 금식 안내 포스터.
    상단 굵게 크게: '검사 전 8시간 금식 필수' (한국어 그대로, 오타·추가 금지),
    물컵에 X 표시 아이콘 중앙,
    파스텔 파란 배경, 깔끔한 플랫 디자인, 1:1 정사각형.
    ```

=== "예약 안내 카드"

    ```
    건강검진 예약 변경 안내 카드,
    달력과 체크 아이콘, 친근한 만화풍,
    밝고 따뜻한 색감, 1:1 카카오톡 카드용.
    ```
