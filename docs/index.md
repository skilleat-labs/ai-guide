## 수업 전 2분 체크리스트

수업 시작 전에 꼭 해주세요!

=== "ChatGPT 로그인"

    1. [chatgpt.com](https://chatgpt.com) 접속
    2. **구글 계정**으로 로그인
    3. 새 채팅 화면이 보이면 성공 ✅

    !!! tip "무료 계정 팁"
        무료는 모델 선택 없이 **GPT-5가 자동 적용**돼요.
        5시간당 약 10개 이후엔 가벼운 버전으로 바뀌니,
        **"한 번에 잘 묻기"** 가 그래서 더 중요합니다.

=== "Notion 로그인"

    1. [notion.so](https://notion.so) 접속
    2. **구글 계정**으로 로그인
    3. 워크스페이스 화면이 보이면 성공 ✅

=== "ChatGPT 한국어 설정"

    1. 좌하단 **프로필 클릭** → **설정**
    2. **일반(General)** 탭 선택
    3. **Language** → **한국어** 선택 후 저장
