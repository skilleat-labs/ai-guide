## 같은 AI에게, 다르게 물어보면

=== "그냥 막 쓰면 ❌"

    **입력:**
    ```
    "중성지방 설명해줘"
    ```

    **결과:**

    - 교과서식 일반 설명, 너무 길다
    - 의학 용어 그대로 → 수검자가 못 알아봄
    - 누구에게 쓸지 모르니 톤이 어정쩡

=== "맥락을 주면 ✅"

    **입력:**
    ```
    "검진 담당자로서, 중성지방 높은 수검자에게,
    의미와 생활수칙을 쉬운 말 5문장으로"
    ```

    **결과:**

    - 수검자가 바로 알아듣는 안내문
    - 딱 5문장, 따뜻한 말투
    - 복붙해서 바로 쓸 수 있는 수준

**차이는 "맥락" 한 끗 — 그 맥락을 빠짐없이 담는 게 4칸 공식입니다.**

## 좋은 답을 받는 4칸 공식

> 이 네 칸만 채우면 됩니다. **오늘 이것만 가져가도 성공.**

| 칸 | 이름 | 내용 | 예시 |
|----|------|------|------|
| 1 | **역할** | 누구로서 답할지 | "건강검진 안내 담당자로서" |
| 2 | **상황** | 어떤 맥락인지 | "중성지방이 높게 나온 수검자에게" |
| 3 | **요청** | 무엇을 해줄지 | "의미와 생활수칙을 안내문으로" |
| 4 | **말투** | 어떤 톤으로 | "쉽고 따뜻한 말투로, 5문장" |

### 4칸 공식 완성 예시

```
역할:  건강검진 안내 담당자로서
상황:  중성지방이 높게 나온 수검자에게
요청:  의미와 생활수칙을 안내문 형식으로 작성해줘
말투:  쉽고 따뜻한 말투로, 5문장 이내로

→ 위 조건으로 수검자 안내문을 작성해줘.
```

!!! tip "4칸 꿀팁"
    - **역할** 칸은 AI에게 "전문가 모드"를 켜는 스위치입니다
    - **말투** 칸에 "글자 수" 또는 "문장 수"를 명시하면 딱 맞게 나와요
    - 결과가 마음에 안 들면 **"더 쉽게"** **"더 짧게"** **"더 따뜻하게"** 라고 한 마디만 추가로 입력하면 됩니다
