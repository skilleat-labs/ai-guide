## 수업에서 못 다룬 꿀팁

### 꿀팁 1 · 프롬프트 시작을 "역할"로 하면 더 좋은 이유

AI는 프롬프트의 **첫 부분을 가장 중요하게** 읽어요.
"역할"을 먼저 쓰면 AI가 그 관점에서 모든 답을 구성합니다.

```
❌ "검진 안내문 써줘 (간호사로서)"
✅ "간호사로서, 검진 안내문을 써줘"
```

---

### 꿀팁 2 · "예시를 보여주고 따라하게 하기"

원하는 스타일의 예시를 직접 보여주면 훨씬 정확하게 나와요.

```
아래 예시처럼 따뜻하고 쉬운 말투로 써줘:

예시:
"이번 검진에서 혈당 수치가 조금 높게 나왔어요.
걱정하실 필요는 없지만, 단 음식을 줄이고
꾸준히 걸으면 좋아질 수 있답니다.
궁금한 점은 언제든 문의해 주세요 😊"

이 톤으로 [다른 검진 항목]에 대한 안내문을 써줘.
```

### 꿀팁 3 · 길이 조절 마법 문장

| 원하는 것 | 입력 |
|----------|------|
| 더 짧게 | `"핵심만 3문장으로 줄여줘"` |
| 더 길게 | `"각 항목을 2문장씩 더 풀어서 써줘"` |
| 딱 맞게 | `"50자 내외로 써줘"` |
| 여러 버전 | `"짧은 버전 / 중간 버전 / 긴 버전으로 각각 써줘"` |

---

### 꿀팁 4 · "다시 써줘" 대신 구체적으로

```
❌ "다시 써줘"  → AI가 뭘 바꿔야 할지 모름

✅ "2번째 문장이 너무 딱딱해. 더 따뜻하게 바꿔줘"
✅ "전체적으로 좋은데 마지막 문장만 더 행동을 유도하게 바꿔줘"
✅ "의학 용어가 아직 남아있어. '혈중 지질'은 쉬운 말로 바꿔줘"
```

---

### 꿀팁 5 · 한 번에 여러 개 요청하기

```
다음 3가지를 한 번에 써줘:
1. 검진 전날 리마인드 문자 (70자 이내)
2. 검진 당일 아침 문자 (50자 이내)
3. 결과 발송 안내 문자 (80자 이내)

모두 친근하고 따뜻한 말투로.
```

!!! tip "한 번에 요청하면 왜 좋냐"
    - 무료 사용 횟수 절약
    - 일관된 톤으로 한 번에 완성
    - 나중에 비교해서 고르기 쉬움

### 꿀팁 6 · "이 문서를 기반으로" 활용법

ChatGPT에 파일을 올린 후:

```
방금 올린 [파일명]을 참고해서
아래 안내문을 작성해줘:
- 대상: [대상]
- 목적: [목적]
- 분량: [분량]
```

!!! warning "주의"
    외부 가이드라인이나 법령을 올릴 때는
    저작권 확인하고, 기관 내부 문서는 비식별화 후 업로드하세요.

### 꿀팁 7 · 역할극(Role-play) 활용

실제 상황을 시뮬레이션해서 더 현실적인 답을 얻는 방법:

```
나는 건강검진 결과에 불안해하는 수검자고,
너는 친절한 검진 상담사야.

수검자: "검사 결과에서 간 수치가 높다고 나왔는데,
         암인가요? 너무 걱정돼요."

상담사 입장에서 안심시키고 다음 단계를 안내하는
답변을 해줘.
```

---

### 꿀팁 8 · 출력 형식 지정하기

같은 내용도 형식이 다르면 쓰임새가 달라집니다.

=== "목록형"
    ```
    검진 준비사항을 번호 목록으로 써줘.
    - 번호는 1.2.3. 형식
    - 각 항목은 한 줄 이내
    ```

=== "표 형식"
    ```
    검진 항목별 준비사항을 표로 정리해줘.
    - 열: 검진 항목 / 준비사항 / 주의사항
    ```

=== "카드 형식"
    ```
    각 정보를 카드처럼 제목 + 내용 2줄 형식으로 써줘.
    예:
    [제목]
    내용 첫 줄
    내용 둘째 줄
    ```

## 이런 말 하면 더 좋아요

### 말투 조절 마법 표현

| 원하는 분위기 | 프롬프트에 추가할 말 |
|-------------|-------------------|
| 따뜻하게 | "걱정을 덜어주는 따뜻한 말투로" |
| 전문적으로 | "신뢰감 있고 전문적인 톤으로" |
| 간결하게 | "핵심만, 군더더기 없이" |
| 쉽게 | "중학생도 이해할 수 있게" |
| 공식적으로 | "공식 문서 형식으로, 존댓말" |
| 친근하게 | "오랜 지인에게 말하듯이" |
| 행동 유도 | "읽는 사람이 바로 행동하게" |

## 실수 방지 체크리스트

!!! danger "이건 꼭 피하세요"
    - [ ] AI가 만든 수치·날짜를 검수 없이 그대로 사용
    - [ ] 수검자 개인정보를 일반 채팅에 입력
    - [ ] 결과를 한 번만 받고 그냥 사용 (최소 한 번은 다듬기)
    - [ ] 너무 긴 프롬프트 (핵심만, 명확하게)
    - [ ] 애매한 요청 ("좋게 써줘" → 뭘 기준으로?)

!!! success "이렇게 하면 잘 됩니다"
    - [ ] 4칸 공식 (역할+상황+요청+말투) 다 채우기
    - [ ] 결과 받은 후 최소 1번 이상 다듬기
    - [ ] 완성된 문서는 바로 노션에 저장
    - [ ] 수치·기준은 직접 확인 후 최종본 발송
    - [ ] 잘 된 프롬프트는 복사해서 보관해두기

## 업무별 프롬프트 모음 (추가)

??? example "회의록 정리"
    ```
    아래는 오늘 회의 내용이야. (회의 내용 붙여넣기)

    이걸 회의록 형식으로 정리해줘:
    - 일시: [날짜]
    - 참석자: [이름들]
    - 주요 논의사항: 핵심 3가지
    - 결정사항: 번호 목록
    - 다음 액션 아이템: 담당자 + 기한 포함
    ```

??? example "이메일 작성"
    ```
    [받는 사람: 기관명 / 직책]에게
    [목적: 단체검진 문의 / 결과 통보 / 협력 제안] 이메일을 써줘.

    - 핵심 내용: [내용]
    - 첨부 파일: [있음/없음]
    - 톤: 정중하고 전문적으로
    - 길이: A4 반 페이지 이내
    - 마무리: 답변 요청으로 끝내기
    ```

??? example "보고서 요약"
    ```
    아래 보고서를 경영진에게 보고하기 위한
    1페이지 요약으로 만들어줘.

    [보고서 내용 붙여넣기]

    요약 형식:
    - 핵심 현황: 3줄
    - 문제점: 2줄
    - 제안 사항: 3가지
    - 기대 효과: 2줄
    ```

??? example "SNS 카피"
    ```
    [기관명] 건강검진센터의 인스타그램에 올릴
    [주제] 관련 게시물 카피를 3가지 버전으로 써줘.

    - 버전 1: 정보 제공형 (건강 팁)
    - 버전 2: 감성형 (공감 메시지)
    - 버전 3: 홍보형 (서비스 소개)

    각 150자 이내, 해시태그 5개, 이모지 적절히 사용.
    ```

??? example "맞춤법·문체 교정"
    ```
    아래 글의 맞춤법을 교정하고,
    더 자연스러운 표현으로 다듬어줘.
    내용은 바꾸지 말고 표현만 개선해줘.

    [원본 글 붙여넣기]
    ```

## 자주 묻는 질문

??? question "프롬프트가 길어도 괜찮나요?"
    **적당히 길게** 는 좋지만 너무 길면 AI가 중요한 걸 놓칠 수 있어요.
    핵심 조건 5개 이내로 명확하게 쓰는 게 더 효과적입니다.

??? question "한글로 써도 되나요, 영어로 써야 하나요?"
    **한글로** 써도 됩니다. 오히려 한국어 업무에는 한글 프롬프트가 더 정확할 수 있어요.

??? question "같은 프롬프트를 다시 쓰면 같은 결과가 나오나요?"
    AI는 매번 조금씩 다른 결과를 줍니다.
    마음에 드는 결과는 반드시 바로 복사해두세요!

??? question "프롬프트를 어디에 저장하면 좋나요?"
    - **노션에 "프롬프트 모음" 페이지** 만들어두기
    - 잘 된 프롬프트는 ChatGPT **프로젝트 지시문** 에 저장
    - 이 가이드 페이지를 북마크해두기
