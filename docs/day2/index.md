## 2주, 한 사이클 완성

```
① 글로 안내 만들고  →  ② 그림으로 보기 좋게  →  ③ 노션 DB에 모아 재사용
```

> **이제 여러분은 AI로 글·그림을 만들고, 노션에 모아 팀과 재사용할 수 있습니다.**
