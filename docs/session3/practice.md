# 3교시 · AI 문서 → 노션 실습

> **STEP 3 (오늘의 핵심) · 직접 해봅니다**

---

## 오늘의 최종 목표

```
① AI로 문서 만들고  →  ② 노션에 쌓고  →  ③ 팀이 재사용한다
```

---

## Step-by-Step 따라하기

### Phase 1 · 노션 팀 페이지 만들기

**Step 1.** 노션 접속 후 새 페이지 만들기
```
왼쪽 사이드바 → "+ 새 페이지"
페이지 제목: "우리 팀 검진 안내 키트"
```

**Step 2.** 하위 페이지 구조 잡기
```
/page → "검진 안내문 모음"
/page → "리마인드 문자 모음"
/page → "FAQ 답변 모음"
```

**Step 3.** 표지 꾸미기 (선택)
```
페이지 상단 → "커버 추가" → 원하는 이미지 선택
"아이콘 추가" → 이모지 선택 (예: 🏥)
```

---

### Phase 2 · AI 문서를 노션에 붙이기

**Step 4.** ChatGPT에서 완성된 안내문 복사
```
마음에 드는 결과를 드래그해서 선택
Ctrl+C (Windows) / Cmd+C (Mac)
```

**Step 5.** 노션에 붙여넣기
```
"검진 안내문 모음" 페이지 열기
빈 공간 클릭 후 Ctrl+V / Cmd+V
```

**Step 6.** 제목 달기
```
붙여넣은 내용 맨 위에 커서 위치
/h1 또는 /제목 입력 → 문서 제목 작성
예: "중성지방 수치 안내문"
```

**Step 7.** 구분선으로 나누기
```
다음 문서를 붙이기 전에
/divider 또는 /구분선 입력 → Enter
```

---

### Phase 3 · 문서 정리하고 보기 좋게

**Step 8.** 날짜·작성자 정보 추가
```
제목 아래에 작은 텍스트로:
작성일: 2026-06-15 | 담당: [부서명]
```

**Step 9.** 콜아웃 박스로 주의사항 강조
```
/callout 입력
"⚠️ 수치는 반드시 직접 확인 후 발송하세요"
```

**Step 10.** 목차 추가 (선택)
```
/toc 또는 /목차 입력
→ 제목들이 자동으로 목차로 정리됩니다
```

---

### Phase 4 · 팀원과 공유

**Step 11.** 팀원 초대
```
페이지 우상단 "공유" 클릭
→ 팀원 이메일 입력
→ 권한: 편집자 또는 보기만
→ 초대 전송
```

**Step 12.** 링크 복사해서 공유
```
공유 → "링크 복사"
→ 카톡이나 메일로 전송
```

---

### Phase 5 · ChatGPT ↔ 노션 연결하고 자동으로 저장하기

> **오늘의 하이라이트 — 복붙 없이 AI가 직접 노션에 씁니다.**

**Step 13.** ChatGPT 설정 → 앱 연결

```
ChatGPT → 좌하단 프로필 → 설정(Settings)
→ 왼쪽 메뉴에서 "앱(Apps)" 클릭
→ 목록에서 "Notion" 찾기 → "연결(Connect)" 클릭
```

**Step 14.** Notion 로그인 및 권한 허용

```
Notion 로그인 창 → 구글 계정으로 로그인
→ 연결할 워크스페이스 확인
→ "허용(Allow)" 클릭 → 연결 완료 ✅
```

!!! warning "연결 안 보이면?"
    ChatGPT 업데이트 시기에 따라 메뉴 이름이 다를 수 있어요.

    - **"Apps"** 탭
    - **"Connections"** 탭
    - **"연결된 앱"** 탭

    셋 중 하나를 찾으면 됩니다. 없으면 강사에게 문의!

**Step 15.** 연결 확인 후 실제로 저장해보기

연결이 됐으면 ChatGPT 채팅창에서 바로 테스트해요.

```
[입력 예시]
방금 만든 중성지방 안내문을
노션 "우리 팀 검진 안내 키트" 페이지에 저장해줘.
페이지 제목은 "중성지방 수치 안내문"으로.
```

**Step 16.** 노션에서 확인

```
notion.so 열기
→ "우리 팀 검진 안내 키트" 페이지 확인
→ "중성지방 수치 안내문" 페이지가 자동 생성됐는지 확인 ✅
```

!!! success "이걸로 오늘 실습 완성!"
    ChatGPT에서 만들고 → 노션에 저장하는 사이클이
    이제 **말 한 마디** 로 끝납니다.

---

### 노션 연결 활용 프롬프트 모음

=== "문서 저장"

    ```
    방금 작성한 [문서 이름]을
    노션 "[페이지 이름]" 페이지 하위에 새 페이지로 저장해줘.
    제목은 "[제목]"으로.
    ```

=== "한 번에 여러 개"

    ```
    오늘 만든 안내문 [개수]개를
    노션 "검진 안내문 모음" 하위에 각각 별도 페이지로 저장해줘.
    각 페이지 제목은 검진 항목명으로.
    ```

=== "내용 추가"

    ```
    노션 "[페이지 이름]" 페이지 맨 아래에
    아래 내용을 추가해줘:

    [추가할 내용]
    ```

=== "노션 내용 읽기"

    ```
    노션 "[페이지 이름]" 페이지의 내용을 보여줘.
    핵심 내용만 요약해줘도 돼.
    ```

=== "전체 워크플로우 한 방에"

    ```
    건강검진 안내 담당자로서,
    중성지방이 높게 나온 수검자에게 보낼
    안내문을 쉬운 말로 5문장으로 작성하고,
    완성되면 노션 "우리 팀 검진 안내 키트" 페이지에
    "중성지방 안내문" 제목으로 저장해줘.
    ```

    !!! tip "이게 진짜 자동화"
        작성 요청 + 노션 저장을 **한 프롬프트** 로 동시에 처리합니다.

## 완성된 페이지 구조 예시

```
🏥 우리 팀 검진 안내 키트
│
├── 📄 검진 안내문 모음
│   ├── 중성지방 수치 안내문         ← 오늘 만든 것 1
│   ├── 혈압 안내문                  ← 오늘 만든 것 2
│   └── 혈당 안내문                  ← 나중에 추가
│
├── 💬 리마인드 문자 모음
│   ├── 검진 3일 전 리마인드 문자    ← 오늘 만든 것 3
│   ├── 미수검 독려 문자
│   └── 재검 안내 문자
│
└── ❓ FAQ 답변 모음
    ├── 금식 관련 FAQ
    ├── 약 복용 관련 FAQ
    └── 결과 확인 FAQ
```

## 노션 유용한 단축키

| 단축키 | 기능 |
|--------|------|
| `/` | 블록 메뉴 열기 |
| `Ctrl+/` | 선택한 블록 타입 변경 |
| `Ctrl+D` | 블록 복제 |
| `Ctrl+Shift+M` | 코멘트 추가 |
| `Ctrl+[` | 들여쓰기 취소 |
| `Ctrl+]` | 들여쓰기 |
| `Ctrl+Enter` | 페이지 열기 (하위 페이지) |
| `@` 입력 | 멤버/페이지/날짜 멘션 |

    - [ ] "우리 팀 검진 안내 키트" 페이지 만들었나요?
    - [ ] 오늘 ChatGPT에서 만든 문서 최소 1개 이상 붙여넣었나요?
    - [ ] 제목(/제목)을 달았나요?

    **자동화 (Phase 5)**

    - [ ] ChatGPT 설정 → 앱 → Notion 연결했나요?
    - [ ] "노션에 저장해줘"로 자동 저장 테스트 해봤나요?
    - [ ] 노션에서 실제로 페이지가 생성됐는지 확인했나요?

    - [ ] 이 가이드 페이지 북마크 했나요?

    오늘 만든 페이지를 꼭 유지해두세요!

### 데이터베이스로 업그레이드

지금은 단순 페이지지만, 나중에 DB로 바꾸면:

```
노션 DB 만들기:
→ /database 입력
→ "안내문 DB" 생성
→ 속성 추가: 항목, 대상, 작성일, 담당자, 상태
```

**결과:** 안내문을 항목별·날짜별·상태별로 필터해서 볼 수 있어요.

## Q&A

??? question "노션이 느려요"
    브라우저보다 **데스크탑 앱** 이 더 빠릅니다.
    notion.so/desktop 에서 다운로드하세요.

??? question "핸드폰에서도 볼 수 있나요?"
    네! 노션 앱(iOS/Android)에서 모바일로 확인 가능합니다.

??? question "실수로 내용을 지웠어요"
    `Ctrl+Z` (실행 취소) 또는
    페이지 우상단 `...` → **"히스토리"** 에서 이전 버전으로 복원 가능합니다.

??? question "팀원이 실수로 내용을 바꿨어요"
    히스토리 기능으로 이전 버전 확인·복원 가능합니다.
    중요한 페이지는 **"잠금"** 기능도 있어요.

??? question "노션 무료로 얼마나 쓸 수 있나요?"
    무료 플랜으로도 **페이지 무제한** 생성 가능합니다.
    팀 협업(게스트 초대)은 10명까지 무료예요.

??? question "ChatGPT 앱 설정에서 Notion이 안 보여요"
    ChatGPT 업데이트 버전에 따라 위치가 다를 수 있어요.
    아래 순서로 찾아보세요:

    1. 설정 → **"앱(Apps)"** 탭
    2. 설정 → **"Connections"** 탭
    3. 설정 → **"연결된 앱"** 탭
    4. 채팅창 왼쪽 **"도구(Tools)"** 또는 **"+"** 아이콘

    그래도 없으면 강사에게 화면 보여주세요!

??? question "Notion 연결 후 어떤 페이지에 접근할 수 있나요?"
    연결 시 허용한 **워크스페이스의 모든 페이지** 에 접근할 수 있어요.
    특정 페이지만 허용하고 싶다면 Notion 설정에서 조절 가능합니다.
    연결을 끊고 싶으면 Notion 설정 → **연결된 앱** 에서 제거하면 됩니다.

??? question "저장 명령을 했는데 노션에 안 들어갔어요"
    몇 가지 원인이 있어요:

    1. **연결이 끊긴 경우** → 설정에서 다시 연결
    2. **페이지 이름이 틀린 경우** → 노션 페이지 정확한 이름으로 다시 요청
    3. **권한이 없는 경우** → 해당 페이지에 편집 권한 확인

    가장 쉬운 해결: `"노션 연결 상태 확인해줘"` 라고 입력해보기

??? question "Notion 연결 기능이 무료인가요?"
    ChatGPT 앱 연결 기능은 **무료 계정에서도 사용 가능**합니다.
    다만 ChatGPT 무료 계정의 사용 횟수 제한 안에 포함되니
    중요한 문서 저장에 집중해서 사용하세요.
