# 2교시 · 직무별 실습 가이드

> 4칸 공식 → 3가지 변형 → 노션 DB에 저장

## 실습 목표

이 시간이 끝나면 내 업무에서 **바로 쓸 수 있는 문서 도우미** 1개를 완성합니다.

## 직무별 프롬프트 모음

=== "종합검사·간호직"

    ### 검진 전 준비 안내

    ```
    건강검진센터 간호 담당자로서,
    종합검진 예약 고객에게
    검진 전날부터 당일까지 꼭 지켜야 할 준비사항을
    번호를 매겨 5가지 이내로 안내해줘.
    - 검진 항목: 혈액검사 + 위내시경 포함
    - 말투: 친절하고 명확하게
    - 강조할 것: 금식 시간, 금지 약물, 준비물
    ```

    ### 검진 결과 안내 (항목별)

    ```
    [검진 항목: 혈압/혈당/콜레스테롤/간수치 중 선택]
    수치가 [정상 범위 초과] 로 나온 수검자분께
    - 이 수치가 무엇을 의미하는지 (전문용어 없이)
    - 생활에서 바로 실천 가능한 수칙 3가지
    - 병원 방문이 필요한 수준인지 여부
    를 쉽고 따뜻하게 안내하는 5문장 이내 안내문을 써줘.
    ```

    ### 미수검 독려 문자

    ```
    건강검진 담당 간호사로서,
    회사 건강검진 기간인데 아직 예약을 안 하신 분께
    보내는 리마인드 문자를 2가지 버전으로 써줘.
    - 버전 1: 부드럽게 독려
    - 버전 2: 마감 임박 긴급 알림
    - 각 70자 이내
    ```

=== "영상의학팀"

    ### 초음파 검사 전 안내

    ```
    영상의학팀 담당자로서,
    복부 초음파 검사를 앞둔 수검자에게
    검사 전 준비사항과 검사 중 협조사항을
    번호를 매겨 안내해줘.
    - 금식 여부, 옷차림, 검사 소요 시간 포함
    - 처음 받는 분도 안심할 수 있도록
    - 5가지 이내
    ```

    ### CT/MRI 검사 전 안내

    ```
    영상의학팀 담당자로서,
    [CT/MRI] 검사를 앞둔 수검자에게
    - 사전 준비사항 (금식, 금속 제거 등)
    - 검사 중 주의사항
    - 조영제 사용 시 주의사항 (해당하는 경우)
    을 안심시키는 톤으로 안내해줘.
    의학 용어는 쉬운 말로 풀어서.
    ```

    ### 검사 결과 설명 초안

    ```
    영상의학팀 담당자로서,
    [갑상선 결절/간 낭종/담석 등] 이 발견된 수검자분께
    - 이게 무엇인지 (겁주지 않고)
    - 대부분의 경우 어떻게 진행되는지
    - 다음에 해야 할 것
    을 3~4문장으로 설명해줘.
    불필요한 불안을 주지 않는 톤으로.
    ```

=== "원무·상담"

    ### 재검 안내 문자

    ```
    건강검진센터 원무팀 담당자로서,
    [검진 항목] 재검이 필요한 고객에게
    재검 안내 문자를 3가지 버전으로 써줘.
    - 버전 1: 일반 재검
    - 버전 2: 정밀 검사 필요
    - 버전 3: 즉시 병원 방문 권고
    각 100자 이내, 전화번호 [○○○-○○○○] 포함.
    ```

    ### 예약 변경 안내

    ```
    건강검진센터 원무팀으로서,
    예약 날짜 변경이 필요한 고객에게 보내는
    카카오 알림톡 메시지를 써줘.
    - 포함 내용: 기존 예약일, 변경 방법, 연락처
    - 톤: 공손하고 친절하게
    - 분량: 150자 이내
    ```

    ### FAQ 답변 모음

    ```
    건강검진센터 상담 담당자로서,
    다음 질문에 대한 답변을 각각 3문장 이내로 써줘.

    1. 검진 당일 아침에 물을 마셔도 되나요?
    2. 위내시경은 수면으로 받을 수 있나요?
    3. 혈압약을 복용 중인데 당일 먹어도 되나요?
    4. 생리 중에 검진을 받을 수 있나요?
    5. 결과는 언제 나오나요?

    각 답변은 친절하고 정확하게, 불안을 주지 않게.
    ```

=== "경영지원·마케팅"

    ### 사내 검진 공지문

    ```
    경영지원팀 담당자로서,
    전 직원 대상 사내 건강검진 실시 공지문을 써줘.
    - 검진 기간: [기간]
    - 검진 장소: [장소]
    - 예약 방법: [방법]
    - 준비사항: 핵심 3가지
    - 톤: 공식적이되 딱딱하지 않게
    - 분량: A4 반 페이지 이내
    ```

    ### SNS 홍보 카피

    ```
    의료기관 마케팅 담당자로서,
    [시즌/이벤트] 건강검진 프로모션 홍보를 위한
    인스타그램 캡션을 3가지 버전으로 써줘.
    - 혜택: [혜택 내용]
    - 기간: [기간]
    - 버전 1: 건강 메시지 중심
    - 버전 2: 혜택 강조
    - 버전 3: 감성적 접근
    각 150자 이내, 해시태그 5개 포함.
    ```

    ### 단체검진 제안서 초안

    ```
    의료기관 영업/마케팅 담당자로서,
    [기업명] 임직원 단체 건강검진 제안서의
    소개 문단과 주요 장점 3가지를 써줘.
    - 강조 포인트: 편의성, 전문성, 비용 효율
    - 톤: 전문적이고 신뢰감 있게
    - 분량: 각 항목 2~3문장
    ```

## 4칸 → 3가지 변형 실습

하나의 주제로 **3가지 다른 버전** 을 뽑는 방법입니다.

### 예시: 재검 안내 문자 3가지 버전

**Step 1.** 첫 번째 버전 작성
```
위의 재검 안내 문자 프롬프트 입력
```

**Step 2.** 2번째 버전 요청
```
같은 내용인데 더 긴급한 느낌으로 다시 써줘.
```

**Step 3.** 3번째 버전 요청
```
이번엔 더 따뜻하고 걱정을 덜어주는 톤으로 써줘.
```

**Step 4.** 3개 중 가장 좋은 것 골라서 노션에 저장

!!! tip "3가지 변형이 필요한 이유"
    같은 상황이어도 대상자에 따라 다른 톤이 필요해요.
    3가지를 만들어두면 상황에 맞게 골라 쓸 수 있어요.
