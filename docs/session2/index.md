## 무료 기능 ① 파일·이미지 업로드

### 어떻게 쓰나

```
채팅창 → 왼쪽 클립(📎) 아이콘 → 파일 선택
→ 파일이 올라가면 질문 입력
```

### 활용 예시

=== "PDF 분석"

    **검진 안내 PDF를 올리고:**
    ```
    이 문서에서 수검자에게 꼭 전달해야 할
    핵심 내용 5가지를 쉬운 말로 요약해줘.
    ```

=== "표·데이터 분석"

    **엑셀/CSV를 올리고:**
    ```
    이 데이터에서 특이한 점이 있으면 알려줘.
    항목별 평균도 계산해줘.
    ```

=== "이미지 분석"

    **안내문 이미지를 올리고:**
    ```
    이 안내문에서 수정이 필요한 부분이 있으면
    구체적으로 알려줘.
    ```

!!! tip "파일 업로드 꿀팁"
    - 무료 계정은 **한 번에 여러 파일** 도 가능해요
    - PDF는 텍스트가 추출 가능한 파일이어야 해요 (스캔본은 안 될 수 있음)
    - 파일이 너무 크면 핵심 페이지만 잘라서 올려도 됩니다

## 무료 기능 ② 프로젝트 = 업무 맥락을 가두는 폴더

### 프로젝트 만들기

```
왼쪽 사이드바 → "프로젝트" → "+ 새 프로젝트"
→ 이름 입력 (예: "검진 안내 문서")
```

### 프로젝트 안에 넣을 수 있는 것

| 항목 | 설명 |
|------|------|
| **관련 대화** | 검진 안내 관련 채팅을 한 폴더에 |
| **전용 지시문** | 이 폴더 안에서만 적용되는 톤·규칙 |
| **참고 파일** | 검진 양식·안내문 PDF (무료 5개) |

!!! success "비유로 이해하기"
    노션이 "결과물 폴더"라면,
    프로젝트는 "대화의 폴더" — 둘 다 맥락을 가두는 같은 개념이에요.

## 무료 기능 ③ GPTs — 만들지 말고, 갖다 쓰기

누군가 만들어 둔 "도우미 GPT"를 검색해서 바로 쓸 수 있어요.

### GPTs 찾는 방법

```
ChatGPT → 왼쪽 사이드바 → "GPT 탐색(Explore GPTs)" → 검색
```

### 추천 GPTs

| GPT 이름 (검색어) | 활용 용도 |
|------------------|----------|
| **PDF 요약** | 긴 자료를 핵심만 추출 |
| **쉬운 말 설명** | 어려운 용어 풀이 |
| **맞춤법·문서 다듬기** | 작성한 문서 교정 |
| **한국어 문서 작성** | 업무 문서 전문 작성 |
| **번역** | 영문 자료 한국어 번역 |

!!! warning "GPTs 사용 주의"
    써드파티 GPTs에는 개인정보나 기관 내부 정보를 입력하지 마세요.
    신뢰할 수 없는 GPT는 OpenAI 외부 서버로 데이터가 전송될 수 있어요.

## 오늘 만들 '도우미' 고르기

한 번 만들어두면 평생 쓰는 자산 — ChatGPT로 만들고, 노션에 쌓습니다.

=== "리마인드 문자 도우미"

    **용도:** 검진 전·후 안내, 미수검 리마인드, 결과 통보

    **특징:** 상황만 바꿔 끝없이 재사용

    **프롬프트 틀:**
    ```
    건강검진센터 담당자로서,
    [상황: 검진 X일 전 / 미수검 / 결과 통보]에
    [대상]에게 보내는 문자를 작성해줘.
    - 포함할 내용: [내용]
    - 분량: 80자 이내
    - 말투: 친근하고 간결하게
    ```

=== "마음 편한 안내 도우미"

    **용도:** 위험군에게 미리 풀어 설명 — 걱정을 덜어주는 사전 안내

    **예시:** 갑상선 결절·간 수치 풀어쓰기

    **프롬프트 틀:**
    ```
    건강검진 안내 담당자로서,
    [검진 항목] 결과에 대해 걱정하는 수검자분께
    안심시키는 설명과 다음 단계를 안내해줘.
    - 공포를 주는 표현 사용 금지
    - 쉬운 말로, 따뜻하게
    - 3~4문장
    ```

=== "다른 아이디어"

    - 비즈니스 메일 도우미
    - 홍보/SNS 카피 도우미
    - 단체검진 제안서 도우미
    - 회의록 정리 도우미
    - 보고서 초안 도우미
