# 1교시 · 4칸 공식 실습

> **직접 해봅니다 — 안내문 1개 만들기**

---

## 실습 순서

- [x] Step 1. 4칸 공식으로 내 업무 안내문을 한 문장씩 입력
- [x] Step 2. AI가 써준 초안을 읽어보기
- [x] Step 3. 어색한 부분은 "더 쉽게" "더 짧게"로 다듬기
- [x] Step 4. 완성되면 짝꿍과 결과 한 번 비교

## 실습용 프롬프트 템플릿

아래 중 하나를 골라 **[ ]** 부분만 바꿔서 붙여넣으세요.

=== "검진 안내문 (기본)"

    ```
    나는 [부서명/직책]입니다.
    [검진 항목명] 수치가 [높게/낮게] 나온 수검자분께
    이 결과의 의미와 일상에서 실천할 수 있는 생활수칙을
    쉽고 따뜻한 말투로 5문장 이내로 안내문을 작성해줘.
    의학 용어는 쓰지 말고, 수검자가 바로 이해할 수 있게 해줘.
    ```

=== "리마인드 문자"

    ```
    나는 건강검진센터 담당자입니다.
    [검진 예약일] 검진 예약이 된 [대상]에게
    검진 [X]일 전에 보내는 리마인드 문자를 작성해줘.
    꼭 챙겨야 할 준비사항 3가지를 포함하고,
    친근하고 간결하게, 80자 이내로 써줘.
    ```

=== "FAQ 답변"

    ```
    나는 건강검진센터 안내 담당자입니다.
    수검자가 "[자주 묻는 질문]"이라고 물어봤을 때
    친절하고 명확하게 답변하는 글을 써줘.
    3문장 이내로, 불안을 덜어주는 따뜻한 말투로.
    ```

=== "부서별 공지문"

    ```
    나는 [부서명] 담당자입니다.
    [공지 내용]을 팀원들에게 알리는 사내 공지문을 작성해줘.
    - 대상: [팀 전체 / 특정 부서]
    - 핵심 내용: [핵심 3가지]
    - 말투: 업무적이되 딱딱하지 않게
    - 분량: 5줄 이내
    ```

## 직무별 실습 예시

??? example "종합검사·간호직 — 중성지방 안내문"

    **프롬프트:**
    ```
    건강검진 안내 담당 간호사로서,
    이번 검진에서 중성지방(TG) 수치가 200 이상으로 높게 나온 수검자분께
    이 수치의 의미와 일상에서 실천 가능한 식이·운동 수칙을
    쉽고 따뜻한 말투로, 5문장 이내 안내문으로 작성해줘.
    의학 전문용어 대신 일상 언어로 풀어서 써줘.
    ```

    **결과 예시:**
    > 이번 검진에서 혈액 속 지방 수치가 조금 높게 나왔어요.
    > 걱정하실 필요는 없지만, 생활 습관을 조금만 바꾸면 좋아질 수 있답니다.
    > 튀긴 음식과 단 음료를 줄이고, 하루 30분 가볍게 걸어보세요.
    > 금주·금연을 실천하시면 수치가 빠르게 개선되는 경우가 많아요.
    > 3개월 후 재검사를 권장드리며, 궁금한 점은 언제든 문의해 주세요. 😊

??? example "영상의학팀 — CT 검사 전 안내"

    **프롬프트:**
    ```
    영상의학팀 담당자로서,
    복부 CT 검사를 앞둔 수검자분께
    검사 전날부터 지켜야 할 준비사항을
    번호를 매겨 쉽게, 5가지 이내로 안내해줘.
    겁먹지 않도록 안심시키는 톤으로.
    ```

??? example "원무·상담 — 재검 안내 문자"

    **프롬프트:**
    ```
    건강검진센터 원무팀 담당자로서,
    이번 검진 결과 재검이 필요한 수검자분께
    재검 안내 문자를 작성해줘.
    - 재검 항목: [항목명]
    - 예약 전화: [번호]
    - 말투: 친절하고 간결하게
    - 분량: 100자 이내
    ```

??? example "경영지원·마케팅 — 검진 프로모션 공지"

    **프롬프트:**
    ```
    경영지원팀 마케팅 담당자로서,
    [시즌] 건강검진 할인 프로모션을 알리는
    사내 공지 또는 SNS 카피를 작성해줘.
    - 혜택: [혜택 내용]
    - 기간: [기간]
    - 대상: [대상]
    - 톤: 밝고 신뢰감 있게
    - 분량: 3줄 이내
    ```

## AI와 대화하며 다듬기

첫 결과가 마음에 안 들면 이렇게 이어서 입력하세요.

| 원하는 변화 | 추가 입력 |
|------------|----------|
| 너무 길다 | `"더 짧게, 3문장으로 줄여줘"` |
| 너무 딱딱하다 | `"더 따뜻하고 친근한 말투로 바꿔줘"` |
| 어려운 말이 있다 | `"의학 용어 없이 중학생도 알 수 있게 바꿔줘"` |
| 형식을 바꾸고 싶다 | `"번호 매겨서 목록 형식으로 바꿔줘"` |
| 버전을 여러 개 보고 싶다 | `"다른 버전으로 3가지 더 써줘"` |
| 특정 부분만 바꾸고 싶다 | `"2번째 문장만 더 구체적으로 바꿔줘"` |

!!! note "꿀팁: 한 채팅에서 계속 대화하기"
    처음부터 완벽하게 받으려 하지 마세요.
    **초안을 받고 → 피드백을 주고 → 다듬는** 과정이
    훨씬 빠르고 자연스럽습니다.

## 수업 중 자주 나오는 질문

??? question "AI가 영어로 답해요"
    프롬프트 맨 끝에 **"한국어로 답해줘"** 를 추가하세요.
    또는 ChatGPT 설정 → 언어를 한국어로 변경하면 기본값이 됩니다.

??? question "결과가 너무 길어요"
    `"3문장으로 줄여줘"` 또는 `"핵심만 2줄로"` 라고 이어서 입력하세요.

??? question "비슷한 내용을 계속 만들어야 하는데 매번 4칸을 써야 하나요?"
    → **[환경설정 가이드](settings.md)** 에서 단축키처럼 저장하는 방법을 알려드려요!

??? question "AI가 잘못된 정보를 줬어요"
    정상입니다. 특히 수치·기준·날짜는 반드시 직접 확인하세요.
    AI는 초안 작성 도구, 최종 검수는 항상 사람이 합니다.
