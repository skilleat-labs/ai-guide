# 1교시 · ChatGPT 환경설정 가이드

> **4칸을 매번 쓰긴 귀찮죠? → 설정에 저장해두면 됩니다.**

---

## 설정 화면 찾기

```
ChatGPT 화면 → 좌하단 프로필 클릭 → 설정(Settings)
→ 개인화(Personalization) 클릭
```

---

## 두 가지 방법

### 방법 1 · 가장 쉬움 — 프리셋 클릭

**설정 경로:** 개인화 → Characteristics (특성)

클릭 한 번으로 켜고 끄는 프리셋들이 있어요.

| 프리셋 | 효과 |
|--------|------|
| **Warm** | 따뜻하고 친근한 말투 |
| **Enthusiastic** | 활기차고 긍정적인 톤 |
| **Headers & Lists** | 제목과 목록으로 정리해서 답함 |
| **Emoji** | 이모지 포함해서 답함 |

!!! tip "추천 조합"
    업무 문서용으로는 **Warm + Headers & Lists** 조합이 딱 좋아요.

---

### 방법 2 · 더 정교함 — 맞춤 지시문 작성

**설정 경로:** 개인화 → Custom instructions (맞춤 지시문)

내 역할·말투·규칙을 글로 한 번 써두면 = **"4칸 공식의 단축키"**

#### 예시 1 · 가장 쉬운 버전 (조건부)

```
업무 문서를 부탁하면 수검자가 알아듣기 쉽게 써줘.
그 외 평소엔 그냥 편하게.
```

!!! info "핵심: "항상 ~" 아니라 "~할 때만 ~""
    "나는 검진 담당자야"를 그냥 박으면 일상 질문까지 업무 톤이 됩니다.
    **"~할 때만"** 조건을 붙이면 업무/일상이 자연스럽게 분리돼요.

#### 예시 2 · 더 구체적인 버전

```
나는 건강검진센터에서 일하는 담당자입니다.

업무 관련 글쓰기를 부탁하면:
- 수검자가 바로 알아들을 수 있는 쉬운 언어 사용
- 의학 전문용어는 일상 언어로 풀어서 표현
- 따뜻하고 안심시키는 톤 유지
- 요청한 길이보다 길게 쓰지 않기
- 수치·기준은 내가 제공한 것만 사용 (임의로 만들지 않기)

일상 대화에서는 편하게 답해줘.
```

#### 예시 3 · 경영지원·마케팅용

```
나는 의료기관 마케팅/경영지원 담당자입니다.

업무 글쓰기를 부탁하면:
- 신뢰감 있고 전문적이되 딱딱하지 않은 톤
- 핵심을 먼저, 부연설명은 나중에
- 행동을 유도하는 CTA(call-to-action) 포함
- SNS는 간결하게, 공지문은 명확하게
- 오탈자 없이 깔끔하게
```

## 방법 1 vs 방법 2, 뭘 써야 하나?

| | 방법 1 (프리셋) | 방법 2 (맞춤 지시문) |
|--|-----------------|---------------------|
| 난이도 | 쉬움 (클릭만) | 보통 (글 작성 필요) |
| 세밀함 | 낮음 | 높음 |
| 추천 대상 | 대부분의 직원 | 업무가 많은 분 |

## 프로젝트 기능 — 업무와 일상 완전 분리

더 깔끔하게 분리하고 싶다면 **"프로젝트"** 기능을 쓰세요.

```
ChatGPT → 왼쪽 사이드바 → "프로젝트" → "+ 새 프로젝트"
```

!!! success "프로젝트의 장점"
    - 관련 대화를 한 폴더에 모아두기
    - **이 폴더 안에서만 적용되는 톤·규칙** 설정 가능
    - 참고 파일(검진 양식·안내문 PDF) 올려두면 계속 참고 (무료 5개)
    - 폴더 밖 일상 질문은 그대로 평범하게

**비유:** 노션이 "결과물 폴더"라면, 프로젝트는 "대화의 폴더" — 둘 다 맥락을 가두는 같은 개념이에요.

## 보안 설정 — 꼭 알아야 할 것

### 데이터 학습 끄기

!!! warning "중요: 업무 데이터 보호"
    ChatGPT 무료 계정은 기본적으로 대화 내용이 학습에 사용될 수 있어요.
    업무 내용을 입력할 때는 반드시 확인하세요.

**경로:** 설정 → Data controls → Improve the model for everyone → **OFF**

### 민감 정보는 임시 채팅 사용

```
ChatGPT → 왼쪽 사이드바 → "임시 채팅(Temporary Chat)"
```

!!! danger "의료기관 필수"
    수검자 이름, 주민번호, 검진 결과 등 **개인정보가 포함된 내용**은
    반드시 **임시 채팅** 에서만 사용하거나,
    이름·번호를 지운 **비식별화** 후 사용하세요.

    임시 채팅은 기억하지 않고 기록도 안 남습니다.

---

## 채팅 운영 팁

### 한 채팅 = 한 주제

주제가 섞이면 답이 흐려져요. 채팅을 '주제별 폴더'처럼 관리하면 답이 일관됩니다.

**새 채팅을 열어야 할 신호:**
- 답이 점점 짧아진다
- 같은 걸 자꾸 다시 묻는 것 같다
- 옆길로 샌다

### 맥락 이어가기

대화 중 맥락을 유지하면서 새 채팅으로 넘기고 싶다면:
응답 옆 `···` → **"새 채팅으로 분기"** 기능 활용

### 기억 관리

**경로:** 설정 → 개인화 → 메모리 → 켜기·끄기·삭제

AI가 기억하면 편하지만, 민감한 업무 맥락이 저장되지 않도록 주기적으로 확인하세요.
