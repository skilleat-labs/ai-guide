# 3교시 · 노션 사용법 + 정리

> **14:50 – 15:30** · 이제 만든 걸 어디에 쌓을까요?

---

## 노션이 뭐예요?

!!! quote "한 줄 정의"
    **노션(Notion) = AI가 만든 결과물을 쌓아두는 "우리 팀의 집"**

새로 배우는 어려운 도구가 아닙니다.
음식을 담는 **"그릇장"** 이라고 생각하세요.
**넣고 · 꺼내고 · 같이 보는 법** 만 익히면 충분합니다.

### 노션이 필요한 이유

=== "한 그릇에 다 담는 도구"

    - 워드(문서) + 엑셀(표) + 위키(공유) = 노션
    - 흩어진 자료를 한 곳에 모아 정리
    - 프로그램 여러 개 옮겨다닐 필요 없음

=== "팀의 기억 창고"

    - 카톡 공지·메일 첨부는 **흘러가요**
    - 노션은 **'쌓이고 검색되는'** 공간
    - 신입도 보고 따라할 수 있는 자료실

---

## 노션, 이 6가지만 하면 됩니다

| 단계 | 이름 | 핵심 동작 | 오늘 중요도 |
|------|------|----------|------------|
| STEP 1 | **블록** | `/` 누르면 메뉴 나와요 | ⭐⭐⭐ |
| STEP 2 | **페이지·중첩** | 폴더 안에 폴더처럼 | ⭐⭐ |
| STEP 3 | **붙여넣기·정리** | AI 문서를 옮기고 보기 좋게 | ⭐⭐⭐⭐⭐ |
| STEP 4 | **공유** | 팀원 초대 → 함께 보는 공간 | ⭐⭐⭐ |
| STEP 5 | **노션 AI** | "요약해줘 / 표로 정리해줘" | ⭐⭐⭐ |
| STEP 6 | **ChatGPT ↔ 노션 연결** | 복붙 없이 AI가 직접 노션에 저장 | ⭐⭐⭐⭐⭐ |

!!! tip "오늘의 핵심"
    **STEP 3 "붙여넣기·정리"** 로 기본을 익히고,
    **STEP 6 "ChatGPT ↔ 노션 연결"** 로 복붙 없는 자동화까지 완성합니다.

---

## STEP 1 · 블록 = 레고

노션의 모든 것은 **"블록"** 하나하나입니다.

### 블록 만드는 법

```
빈 줄에서 / 를 누르면 메뉴가 떠요
```

### 자주 쓰는 블록 명령어

| 입력 | 블록 종류 |
|------|----------|
| `/h1` 또는 `/제목` | 큰 제목 |
| `/h2` 또는 `/소제목` | 중간 제목 |
| `/bullet` | 글머리 목록 |
| `/number` | 번호 목록 |
| `/todo` | 체크리스트 |
| `/table` | 표 |
| `/divider` 또는 `/구분선` | 구분선 |
| `/quote` | 인용구 (강조 박스) |
| `/callout` | 콜아웃 박스 |
| `/page` | 페이지 안에 새 페이지 |

!!! tip "가장 많이 쓰는 3가지"
    `/ + 제목` → `/ + 글머리` → `/ + 구분선`
    이 세 가지만 알아도 오늘 실습은 충분합니다.

---

## STEP 2 · 페이지·중첩

페이지 안에 또 페이지를 넣을 수 있어요.

```
"우리 팀 검진 키트" (메인 페이지)
├── 검진 안내문 모음 (하위 페이지)
│   ├── 혈압 안내문
│   ├── 혈당 안내문
│   └── 중성지방 안내문
├── 리마인드 문자 모음 (하위 페이지)
└── FAQ 답변 모음 (하위 페이지)
```

**폴더 안에 폴더 — 그 느낌 그대로입니다.**

### 새 페이지 만들기

```
왼쪽 사이드바 → "+ 새 페이지" 클릭
또는 페이지 안에서 /page 입력
```

---

## STEP 3 · AI 문서를 노션에 붙이고 정리하기 (핵심!)

### 4단계 실습

```
1. ChatGPT에서 만든 안내문을 복사 (Ctrl+C / Cmd+C)
2. 노션 팀 페이지에 붙여넣기 (Ctrl+V / Cmd+V)
3. 맨 윗줄은 제목(/제목)으로 키우기
4. 구분선(/구분선)으로 문서끼리 나누기
```

!!! success "이 동작 하나로"
    오늘 만든 모든 문서가 노션 한 페이지에 쌓입니다.
    이게 **"우리 팀 안내 키트"** 의 시작이에요.

### 붙여넣기 팁

!!! tip "서식 없이 붙여넣기"
    ChatGPT에서 복사한 내용을 노션에 붙여넣으면
    기존 서식이 따라올 수 있어요.
    **서식 없이** 붙여넣으려면:
    `Ctrl+Shift+V` (Windows) / `Cmd+Shift+V` (Mac)

---

## STEP 4 · 공유

### 팀원 초대하기

```
페이지 오른쪽 위 → "공유" 버튼 클릭
→ 팀원 이메일 입력
→ 권한 설정 (편집자 / 보기만)
→ 초대 완료
```

이제 팀이 함께 보고 쓰는 **공용 공간** 이 생겼어요.

### 링크로 공유하기

```
공유 → "링크 복사" → 카카오톡이나 메일로 전송
```

!!! warning "권한 주의"
    - **편집자:** 내용을 바꿀 수 있음
    - **보기만:** 읽기만 가능
    - 팀 내부 문서는 "워크스페이스 공유"만, 외부 공개는 신중하게

---

## STEP 5 · 노션 AI

### 어떻게 쓰나

```
페이지 안에서 → 스페이스바 또는 AI 버튼 클릭
```

### 자주 쓰는 명령

| 명령 | 결과 |
|------|------|
| `요약해줘` | 긴 내용을 핵심만 |
| `표로 정리해줘` | 산문을 표로 변환 |
| `더 쉽게 바꿔줘` | 어려운 내용을 쉽게 |
| `제목 만들어줘` | 적절한 제목 제안 |
| `액션 아이템 뽑아줘` | 해야 할 일 목록 |

!!! info "연결·설정 0단계"
    노션 AI는 별도 설정 없이 페이지 안에서 바로 사용 가능합니다.
    (유료 기능이지만 무료 체험 횟수가 있어요)

## STEP 6 · ChatGPT ↔ 노션 연결 (오늘의 하이라이트!)

> **복붙 없이, ChatGPT가 직접 노션에 써줍니다.**

### 어떤 기능인가요?

ChatGPT 설정에서 노션을 연결해두면, 채팅 중에 **"이 내용을 노션에 저장해줘"** 한 마디로
AI가 직접 노션 페이지에 내용을 써줍니다.

```
기존 방식:  ChatGPT에서 작성 → 복사 → 노션 열기 → 붙여넣기 → 정리
연결 후:    ChatGPT에서 작성 → "노션에 저장해줘" → 끝!
```

!!! success "오늘의 최종 목표"
    이 기능까지 완성하면 **"우리 팀 검진 안내 키트"** 가
    클릭 한 번으로 자동으로 쌓이는 구조가 됩니다.

---

### 연결 방법 (설정 3단계)

**Step 1.** ChatGPT 설정 열기
```
ChatGPT → 좌하단 프로필 클릭 → 설정(Settings)
```

**Step 2.** 앱(Apps) 메뉴 진입
```
설정 창 왼쪽 메뉴 → "앱(Apps)" 또는 "연결된 앱" 클릭
```

!!! note "메뉴 위치 안내"
    ChatGPT 업데이트에 따라 메뉴 이름이 다를 수 있어요.
    **"Apps" / "Connections" / "연결"** 중 하나를 찾으면 됩니다.
    (스크린샷 참고 — 이미지 추가 예정)

**Step 3.** Notion 추가
```
앱 목록에서 "Notion" 찾기 → "연결(Connect)" 클릭
→ Notion 로그인 창 뜨면 구글 계정으로 로그인
→ "허용(Allow)" 클릭 → 연결 완료 ✅
```

!!! warning "권한 범위 확인"
    연결 시 Notion 워크스페이스 접근 권한을 요청합니다.
    **어떤 워크스페이스에 접근할지 범위를 확인**하고 허용하세요.
    연결 후 언제든 Notion 설정에서 연결을 끊을 수 있어요.

---

### 연결 후 사용법

연결이 완료되면 ChatGPT 채팅창에서 바로 노션 관련 명령을 쓸 수 있어요.

**노션 페이지에 내용 저장하기:**
```
방금 만든 중성지방 안내문을 노션의
"우리 팀 검진 안내 키트" 페이지에 저장해줘.
제목은 "중성지방 수치 안내문"으로.
```

**노션 페이지 내용 불러오기:**
```
노션의 "검진 안내문 모음" 페이지 내용을 보여줘.
```

**노션 페이지 수정하기:**
```
노션의 "중성지방 수치 안내문" 마지막에
"문의: 검진센터 ○○○-○○○○" 문장을 추가해줘.
```

!!! tip "한 번에 여러 개 저장하기"
    ```
    오늘 만든 안내문 3개를 각각 별도 페이지로
    노션 "검진 안내문 모음" 하위에 저장해줘.
    각 페이지 제목은 검진 항목명으로 해줘.
    ```
    → 3개 페이지가 한 번에 생성됩니다!

---

### 연결 전 vs 연결 후 비교

| | 연결 전 | 연결 후 |
|--|---------|---------|
| 문서 저장 | 복사 → 노션 열기 → 붙여넣기 → 정리 | "노션에 저장해줘" 한 마디 |
| 여러 문서 | 반복 작업 필요 | 한 번에 여러 페이지 생성 |
| 수정 | 노션 열어서 직접 편집 | ChatGPT에서 수정 요청 |
| 팀 공유 | 공유 링크 따로 발송 | 노션 자체 공유 기능 그대로 |

## 데이터베이스 살짝 맛보기 (데모)

### 데이터베이스 = 똑똑한 표

같은 자료를 **여러 모양** 으로 볼 수 있어요.

```
표 뷰     → 검진 항목별 비교
갤러리 뷰  → 카드로 한눈에
캘린더 뷰  → 시즌별 안내
```

### 검진 안내문 DB 예시

| 항목 | 대상 | 발송 시점 | 담당자 | 상태 |
|------|------|----------|--------|------|
| 중성지방 안내 | 수치 200+ | 결과 발송 시 | 간호팀 | 완료 |
| 재검 안내 | 재검 필요자 | 결과 발송 후 2일 | 원무팀 | 준비 중 |
| 검진 전 리마인드 | 예약자 전체 | 검진 3일 전 | 상담팀 | 진행 중 |
